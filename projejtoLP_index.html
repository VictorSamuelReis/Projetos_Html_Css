O HTML

HTML (Linguagem de Marcação de HiperTexto) é o bloco de construção mais básico da web. Define o significado e a estrutura do conteúdo da web. Outras tecnologias além do HTML geralmente são usadas para descrever a aparência/apresentação (CSS) ou a funcionalidade/comportamento (JavaScript) de uma página da web.

O CSS

CSS (Cascading Style Sheets) é uma linguagem de estilo usada para descrever a apresentação de um documento escrito em HTML ou em XML (incluindo várias linguagens de marcação como SVG, MathML ou XHTML). O CSS descreve como os elementos devem ser renderizados na tela, no papel, na fala ou em outros meios.

O JavaScript

JavaScript é uma linguagem de programação que permite implementar funcionalidades mais complexas em páginas da web. Sempre que uma página da web faz mais do que simplesmente mostrar informações estáticas para você - exibe atualizações de conteúdo em um relógio em tempo real, mapas interativos ou conteúdo de vídeo em tela cheia, por exemplo - você pode apostar que o JavaScript provavelmente está envolvido.
